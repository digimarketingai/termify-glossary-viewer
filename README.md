## ✨ Features

### 📥 Import & Export
- **Drag & drop CSV import** - Simply drop your terminology file
- **Export to CSV** - Save your glossary in spreadsheet format
- **Export to JSON** - Full data backup with settings
- **Export to Standalone HTML** - Self-contained shareable glossary page

### 🔍 View & Search
- **Instant search** - Filter terms in real-time
- **Category filtering** - Browse by terminology category
- **Sortable columns** - Click headers to sort
- **Enlarge view** - Click any entry to see full details with image

### ✏️ Edit & Customize
- **Add/Edit/Delete terms** - Full CRUD operations
- **Add image URLs** - Visual reference for each term
- **Custom branding** - Set title, subtitle, logo, and author name
- **6 color themes** - Ocean, Sunset, Forest, Midnight, Cherry, Golden

### 🌐 Bilingual Interface
- **English & Traditional Chinese** - All labels in both languages
- **Perfect for localization teams** - Built by translators, for translators

## 📖 Usage

### Quick Start

1. **Open** the [Termify app](https://digimarketingai.github.io/termify-glossary-viewer/)
2. **Drag & drop** your CSV file (or click to browse)
3. **Search, filter, and sort** your terminology
4. **Click any term** to enlarge and view details
5. **Export** as HTML, CSV, or JSON

### Adding/Editing Terms

1. Click **"➕ Add Term"** to create new entries
2. Click the **✏️ button** on any row to edit
3. Add **image URLs** to include visual references
4. Click **🗑️ Delete** to remove entries

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `ESC` | Close modal |
| `←` `→` | Navigate between terms (in enlarge view) |
| `↑` `↓` | Navigate between terms (in enlarge view) |
